# 快速开始指南 - 物品图片上传功能

## 🚀 5分钟快速体验

### 前置条件
- ✅ 开发服务器已启动：http://localhost:3001
- ✅ 浏览器：Chrome/Edge/Firefox/Safari
- ✅ 准备一张测试图片（任意格式，建议 1-5MB）

---

## 📱 第一步：创建测试数据

### 1. 访问应用
```
打开浏览器，访问：http://localhost:3001
```

### 2. 创建房间和容器（如果还没有）
1. 点击首页的"房间"按钮
2. 点击"客厅"进入房间
3. 点击"上柜-左格"进入容器

### 3. 添加物品
1. 点击右上角"+"按钮
2. 输入物品名称：例如"无线鼠标"
3. 设置数量：1
4. 点击"添加"

---

## 📸 第二步：上传图片

### 方法 1：桌面浏览器
1. 点击刚创建的物品进入详情页
2. 向下滚动到图片区域
3. 点击"上传图片"按钮
4. 选择一张图片（建议：电子产品照片）
5. 等待 1-3 秒，图片自动压缩并显示

### 方法 2：移动设备
1. 手机浏览器访问：http://192.168.31.215:3001
2. 进入物品详情页
3. 点击"上传图片"
4. 选择"拍照"或"从相册选择"
5. 拍摄或选择图片
6. 等待上传完成

---

## 🔍 第三步：查看和管理图片

### 查看大图
1. 在物品详情页，点击图片
2. 全屏查看器打开
3. 点击右上角"X"或背景关闭

### 替换图片
1. 点击图片右上角的"上传"图标（↑）
2. 选择新图片
3. 旧图片自动被替换

### 删除图片
1. 点击图片右上角的"删除"图标（🗑️）
2. 图片立即删除
3. 显示"暂无图片"占位符

---

## 📋 第四步：查看列表缩略图

1. 返回容器页面（点击左上角返回按钮）
2. 观察物品列表
3. 有图片的物品显示缩略图（48px 方形）
4. 无图片的物品显示默认图标

---

## 🔧 常见问题

### Q1: 图片上传失败怎么办？
**A:** 检查以下几点：
1. 文件大小是否 ≤10MB
2. 文件格式是否为图片（JPEG/PNG/WebP）
3. 浏览器控制台是否有错误信息
4. 尝试刷新页面重试

### Q2: 图片不显示怎么办？
**A:** 尝试以下步骤：
1. 刷新页面（F5）
2. 清除浏览器缓存
3. 检查 IndexedDB 是否启用
4. 打开 DevTools → Application → IndexedDB → SpotItDB → images

### Q3: 图片压缩后不清晰怎么办？
**A:** 这是正常现象：
- 压缩算法会降低图片质量以达到 500KB 限制
- 建议上传清晰度较高的原图
- 压缩后的图片仍然足够清晰用于识别物品

### Q4: 移动端相机无法调用怎么办？
**A:** 检查以下几点：
1. 浏览器是否有相机权限
2. 是否使用 HTTPS 或 localhost
3. 尝试使用"从相册选择"代替

## 💡 提示

### 最佳实践
1. **图片选择**：选择清晰、光线充足的照片
2. **文件大小**：建议 1-5MB，压缩效果最佳
3. **拍摄角度**：正面拍摄，物品居中
4. **背景简洁**：避免杂乱背景

### 性能优化
1. 避免同时上传大量图片
2. 定期清理不需要的图片
3. 使用合适的图片尺寸
