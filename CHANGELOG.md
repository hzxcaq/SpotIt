# 更新日志

## [未发布] - 2026-01-20

### ✨ 新增功能

#### 主题系统
- 新增三套现代化主题：海洋蓝（默认）、日落橙、森林绿
- 支持在设置页面实时切换主题
- 主题选择自动保存，下次访问自动应用
- 平滑的主题切换动画（0.3s ease）
- 每个主题都有独特的配色方案和视觉风格

#### 流畅添加体验
- 添加物品时可直接创建新房间，无需跳转到房间管理页面
- 添加物品时可直接创建新容器，无需跳转到容器管理页面
- 创建后自动选中新建的房间/容器，流程更流畅

### 🎨 UI 改进
- 更新为现代化的配色方案，摒弃老旧的灰色调
- 采用高饱和度的主色调，提升视觉吸引力
- 更新系统字体栈，支持更好的跨平台一致性
- 增加圆角半径（0.625rem → 0.75rem），更加圆润

### 🔧 技术改进
- 使用 CSS 自定义属性实现主题切换，性能优异
- 采用 HSL 色彩空间，便于创建协调的配色方案
- 新增主题系统架构：
  - `lib/themes/types.ts` - 主题类型定义
  - `lib/themes/config.ts` - 主题配置
  - `lib/themes/theme-provider.tsx` - 主题 Context
  - `components/theme-selector.tsx` - 主题选择器组件

### 📝 文档更新
- 更新 README.md，添加主题切换和流畅添加功能说明
- 更新项目结构文档，包含主题系统相关文件
