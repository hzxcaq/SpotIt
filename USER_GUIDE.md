# SpotIt 用户指南

欢迎使用 SpotIt - 您的智能物品管理助手！

## 📱 什么是 SpotIt？

SpotIt 是一款帮助您管理家中物品位置的应用。再也不用为找不到东西而烦恼！

**适用场景**：
- 🏠 管理多套房产的物品
- 📦 整理收纳箱、储物柜中的物品
- 🔍 快速找到某件物品的位置
- 📸 为物品拍照，方便识别
- 🔄 记录物品借出、归还情况

---

## 🔒 您的数据，您做主

### 数据存储在哪里？

**重要说明**：您的所有数据都保存在**您自己的设备**上，不会上传到任何服务器。

```
您的数据存储位置：
├── 浏览器本地存储（IndexedDB）
│   └── 所有物品、房间、容器信息
│
└── 您选择的备份文件夹（可选）
    └── 备份文件（JSON 格式）
```

### 这意味着什么？

✅ **隐私安全**
- 您的物品清单、照片、备注等信息完全私密
- 没有任何人（包括我们）能看到您的数据
- 不会被用于广告或其他商业用途

✅ **完全控制**
- 您可以随时导出数据
- 您可以随时删除数据
- 您可以备份到任何地方（电脑、U盘、云盘）

✅ **离线可用**
- 断网也能正常使用
- 数据读写速度快
- 不消耗流量

⚠️ **需要注意**
- 清除浏览器数据会删除所有信息
- 更换设备需要手动导入备份
- 建议定期备份数据到安全位置

## 🚀 快速开始

### 第一步：创建地点

1. 打开 SpotIt 首页
2. 点击顶部的"选择地点"
3. 点击"管理" → "添加新地点"
4. 输入地点名称（如"我的家"、"办公室"）
5. 系统会自动创建 5 个默认房间：
   - 客厅
   - 厨房
   - 主卧
   - 次卧
   - 卫生间

**提示**：每个房间会自动创建 6 个容器（上柜-左/中/右格，下柜-左/中/右格）

### 第二步：添加物品

**方式一：拍照添加（推荐）**
1. 首页点击"拍照添加"
2. 允许浏览器访问摄像头
3. 对准物品拍照
4. 选择房间和容器
5. 填写物品名称、数量等信息
6. 点击"创建物品"

**方式二：手动添加**
1. 首页点击"按房间浏览"
2. 选择房间 → 选择容器
3. 点击"新增物品"
4. 填写物品信息
5. 可选择上传照片
6. 点击"创建"

### 第三步：搜索物品

1. 首页搜索框输入物品名称
2. 查看搜索结果
3. 点击物品查看详细位置
4. 根据位置信息找到物品

## 📚 功能详解

### 1. 地点管理

**什么是地点？**
地点是最高层级的分类，用于区分不同的房产或场所。

**使用场景**：
- 🏠 家里
- 🏢 办公室
- 🏘️ 老家
- 🏖️ 度假屋

**如何管理**：
- 首页顶部点击地点名称可快速切换
- 点击"管理"进入地点管理页面
- 可以添加、编辑、删除地点

### 2. 房间管理

**什么是房间？**
房间是地点下的第二层级，代表实际的房间。

**如何管理**：
1. 首页点击"按房间浏览"
2. 查看当前地点的所有房间
3. 点击"+"添加新房间
4. 点击房间名称进入房间详情
5. 可以编辑或删除房间

**提示**：删除房间会同时删除该房间下的所有容器和物品，请谨慎操作！

### 3. 容器管理

**什么是容器？**
容器是房间内的具体存储位置，如柜子、抽屉、收纳箱等。

**如何管理**：
1. 进入房间详情页
2. 查看该房间的所有容器
3. 点击"+"添加新容器
4. 点击容器查看内部物品
5. 可以编辑或删除容器

**命名建议**：
- 上柜-左格、上柜-中格、上柜-右格
- 抽屉1、抽屉2、抽屉3
- 红色收纳箱、蓝色收纳箱
- 使用清晰易懂的名称

### 4. 物品管理

**物品信息包括**：
- 📝 名称（必填）
- 🔢 数量和单位
- 📸 照片（可选）
- 🏷️ 标签（可选，用逗号分隔）
- 📄 备注（可选）

**物品状态**：
- ✅ 在库：物品在容器中
- 📤 已借出：借给他人
- 🔄 已归还：从他人处归还
- ❌ 已消耗：已使用完
- 🗑️ 已处置：已丢弃

**如何编辑物品**：
1. 点击物品进入详情页
2. 点击"编辑"按钮
3. 修改信息后点击"保存"

**如何移动物品**：
1. 进入物品详情页
2. 点击"移动"按钮
3. 选择新的房间和容器
4. 系统会自动记录移动历史

**如何删除物品**：
1. 进入物品详情页
2. 点击"删除"按钮
3. 确认删除（此操作不可撤销）

### 5. 搜索功能

**搜索范围**：
- 物品名称
- 物品别名
- 物品标签

**搜索技巧**：
- 输入关键词即可，不区分大小写
- 支持模糊搜索
- 搜索结果显示物品所在位置

**示例**：
- 搜索"充电器"：找到所有充电器
- 搜索"工具"：找到标签包含"工具"的物品
- 搜索"螺丝"：找到名称包含"螺丝"的物品

### 6. 历史记录

**记录内容**：
- 物品创建
- 物品移动
- 物品借出
- 物品归还
- 物品更新
- 物品删除

**如何查看**：
1. 首页显示最近 4 条操作记录
2. 点击"查看全部"查看完整历史
3. 每条记录显示操作时间和详情

## 💾 数据备份与恢复

### 为什么要备份？

**数据可能丢失的情况**：
- ❌ 清除浏览器缓存/数据
- ❌ 浏览器更新或重装
- ❌ 系统重装
- ❌ 更换设备
- ❌ 误删除数据

**备份的好处**：
- ✅ 数据安全有保障
- ✅ 可以恢复到任意备份点
- ✅ 可以在不同设备间同步数据
- ✅ 可以长期保存数据

### 自动备份（推荐）

**如何开启**：
1. 进入"设置"页面
2. 找到"自动备份"区域
3. 点击"开启"按钮
4. 选择备份方式

**备份方式一：提示下载**
- 适用于所有浏览器
- 每天首次访问时提示下载
- 备份文件保存到浏览器下载文件夹
- 需要点击一次"立即下载"

**备份方式二：自动保存到文件夹（推荐）**
- 仅支持 Chrome 86+ 和 Edge 86+
- 完全自动化，无需任何操作
- 首次需要选择备份文件夹
- 每天自动保存最新备份

**如何选择备份文件夹**：
1. 开启自动备份
2. 选择"自动保存到文件夹"
3. 浏览器弹出文件夹选择器
4. 选择一个文件夹（建议选择云盘同步文件夹）
5. 点击"允许"授权
6. 完成设置

**推荐的备份位置**：
- 📁 D:\OneDrive\SpotIt备份\
- 📁 D:\Dropbox\SpotIt备份\
- 📁 D:\Google Drive\SpotIt备份\
- 📁 D:\我的文档\SpotIt备份\

**提示**：选择云盘同步文件夹，备份会自动上传到云端，更加安全！

### 手动备份

**如何导出数据**：
1. 进入"设置"页面
2. 找到"手动备份与恢复"区域
3. 点击"导出到文件"
4. 浏览器会下载一个 JSON 文件
5. 将文件保存到安全位置

**备份文件说明**：
- 文件名：spotit-backup-20260118.json
- 文件格式：JSON（纯文本，可用记事本打开）
- 文件内容：所有地点、房间、容器、物品数据
- 文件大小：通常 1-10 MB（取决于物品数量和照片）

### 恢复数据

**方式一：从备份历史恢复**
1. 进入"设置"页面
2. 找到"备份历史"区域
3. 选择要恢复的备份
4. 点击"恢复"按钮
5. 确认操作
6. 等待恢复完成（页面会自动刷新）

**方式二：从文件恢复**
1. 进入"设置"页面
2. 点击"从文件恢复"
3. 选择之前导出的 JSON 文件
4. 确认操作
5. 等待恢复完成（页面会自动刷新）

**注意事项**：
- ⚠️ 恢复数据会清空当前所有数据
- ⚠️ 此操作不可撤销
- ⚠️ 建议先导出当前数据再恢复
- ⚠️ 确保备份文件完整且未损坏

### 备份管理

**浏览器备份**：
- 自动保留最近 7 天的备份
- 保存在浏览器本地存储中
- 可以在"备份历史"中查看
- 可以下载、恢复或删除

**文件备份**：
- 自动保存模式：只保留最新 1 份
- 文件名：spotit-backup-latest.json
- 每天自动覆盖旧文件
- 建议定期手动复制到其他位置

## 🔄 跨设备使用

### 场景一：在新设备上使用

**步骤**：
1. 在旧设备上导出数据
2. 将备份文件传输到新设备（U盘、云盘、邮件等）
3. 在新设备上打开 SpotIt
4. 进入设置 → 从文件恢复
5. 选择备份文件
6. 完成数据迁移

### 场景二：在多个设备间同步

**方法一：手动同步**
1. 在设备 A 上导出数据
2. 上传到云盘
3. 在设备 B 上从云盘下载
4. 导入数据

**方法二：自动同步（推荐）**
1. 在设备 A 上开启自动备份
2. 选择云盘同步文件夹作为备份位置
3. 在设备 B 上从该文件夹导入数据
4. 每天在设备 A 上的更新会自动同步到云盘
5. 定期在设备 B 上导入最新备份

### 场景三：更换浏览器

**步骤**：
1. 在旧浏览器中导出数据
2. 在新浏览器中打开 SpotIt
3. 导入数据
4. 重新设置自动备份

## 💡 使用技巧

### 1. 高效整理物品

**建议流程**：
1. 先创建地点和房间
2. 按房间逐个整理
3. 边整理边拍照录入
4. 使用标签分类（如"工具"、"电子"、"常用"）
5. 重要物品添加详细备注

### 2. 标签使用技巧

**推荐标签**：
- 按类别：工具、电子、文具、衣物、食品
- 按用途：常用、备用、季节性、应急
- 按状态：新品、待修、待处理
- 按所有者：爸爸、妈妈、孩子

**标签格式**：
- 多个标签用逗号分隔
- 例如：工具, 电子, 常用

### 3. 照片拍摄建议

**拍照技巧**：
- 📸 光线充足，避免模糊
- 📸 物品居中，背景简洁
- 📸 多个同类物品可以拍在一起
- 📸 特殊标记或型号要拍清楚

**照片用途**：
- 快速识别物品
- 记录物品外观
- 区分相似物品
- 借出时作为凭证

### 4. 定期维护

**建议每月做一次**：
1. 检查物品状态是否准确
2. 更新已消耗或处置的物品
3. 整理借出未归还的物品
4. 清理不再需要的记录
5. 导出一次备份到电脑

### 5. 借出管理

**借出物品时**：
1. 进入物品详情页
2. 记录借给谁
3. 系统自动标记为"已借出"
4. 可以在历史记录中查看

**归还物品时**：
1. 更新物品状态为"已归还"
2. 系统自动记录归还时间

## ❓ 常见问题

### 数据安全相关

**Q: 我的数据会被上传到服务器吗？**
A: 不会。所有数据都保存在您的设备上，不会上传到任何服务器。

**Q: 你们能看到我的数据吗？**
A: 不能。我们无法访问您的任何数据，包括物品清单、照片等。

**Q: 数据会被用于广告吗？**
A: 不会。我们不收集任何用户数据，也不会用于任何商业用途。

**Q: 清除浏览器数据会删除我的物品记录吗？**
A: 是的。清除浏览器数据会删除所有信息，请务必先备份。

### 备份相关

**Q: 自动备份会占用很多空间吗？**
A: 不会。备份文件通常只有 1-10 MB，即使有很多照片也不会超过 50 MB。

**Q: 我可以在哪里找到备份文件？**
A:
- 提示下载模式：浏览器默认下载文件夹
- 自动保存模式：您选择的备份文件夹

**Q: 备份文件可以用记事本打开吗？**
A: 可以。备份文件是 JSON 格式的纯文本文件，可以用任何文本编辑器打开查看。

**Q: 我可以编辑备份文件吗？**
A: 技术上可以，但不建议。手动编辑可能导致文件损坏，无法恢复。

### 使用相关

**Q: 可以在多个设备上同时使用吗？**
A: 可以，但数据不会自动同步。需要手动导出导入数据，或使用云盘同步备份文件。

**Q: 离线可以使用吗？**
A: 可以。首次访问后，应用会缓存到本地，断网也能正常使用。

**Q: 支持哪些浏览器？**
A: 支持所有现代浏览器：
- ✅ Chrome（推荐）
- ✅ Edge（推荐）
- ✅ Firefox
- ✅ Safari
- ✅ 移动端浏览器

**Q: 可以添加多少个物品？**
A: 理论上没有限制，但建议不超过 10,000 个物品，以保证性能。

**Q: 照片会被压缩吗？**
A: 会。为了节省空间，照片会自动压缩到 800x800 像素，质量为 80%。

### 功能相关

**Q: 可以导出为 Excel 吗？**
A: 目前不支持。备份文件是 JSON 格式，可以使用在线工具转换为 Excel。

**Q: 可以打印物品清单吗？**
A: 可以。在浏览器中使用打印功能（Ctrl+P）即可。

**Q: 可以分享给家人使用吗？**
A: 可以。导出备份文件发送给家人，他们导入后就能看到所有数据。

**Q: 支持多人协作吗？**
A: 目前不支持实时协作。可以通过定期同步备份文件的方式实现简单协作。
